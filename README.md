# Moxie Programming Language

**A consistent, explicit, and pragmatic evolution of Go**

Moxie is a programming language that addresses Go's design inconsistencies while preserving its philosophy of simplicity. The Moxie transpiler converts `.x` files to standard Go code, allowing full interoperability with the Go ecosystem.
